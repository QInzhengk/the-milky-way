# 方法一（快捷简单）
打开指定的文件夹，在路径栏里点击并输入“cmd”，回车，就进入控制台了。默认路径就是指定文件夹的路径。

![在这里插入图片描述](/0af780b561c741ca821d87062492c2c2.png)


# 方法二
在搜索框中搜索“cmd”或按win+R键，打开命令窗口。在默认路径后输入“cd”，空格，再输入（复制）指定文件夹路径。我们发现回车后暂时还是原来默认的路径。
![在这里插入图片描述](/22f71f5b2b17411cb4e337be5a63af2f.png)

然后输入指定文件夹所在的盘，这里我们输入的是“E:"（双引号里的），再回车就是指定的路径了。
![在这里插入图片描述](/efb995a3cb084d1e9bd367e8c3a217db.png)

# 方法三
打开指定的文件夹，按住shift键，在空白处右击，在菜单栏中选择“在此处打开Powershell窗口”。

![在这里插入图片描述](/6ee9b731eee343be840503b6e327d1b3.png)

打开的命令窗口里就是当前路径了。
![在这里插入图片描述](/c0b35f35192a46a98274f19de2b8feed.png)

或者你也可以 先输入路径所在磁盘，例："e:"或"E:"，进入所在磁盘，然后输入cd+空格，一层一层打开文件夹。

![在这里插入图片描述](/7822be7479b440bca32fd16531258d23.png)
